How to Install CasaOS

1. Log into Debian.

2. Install CURL

  By typing sudo apt install curl 

3. Install CasaOS

  By typing curl -fsSL https://get.casaos.io | sudo bash

After that, navigate to the ip of the Debian PC. It will pop up on the screen once it is installed.

Now, set up CasaOS. I think you don't need a tutorial for that :)
